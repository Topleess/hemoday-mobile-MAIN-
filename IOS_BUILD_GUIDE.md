# Руководство по сборке и публикации iOS приложения (Hemoday)

Ваше приложение подготовлено для сборки под iOS. Так как вы работаете на Windows, финальную сборку (`.ipa` файл) необходимо выполнить на компьютере [Mac](https://www.apple.com/mac/).

## 1. Подготовка
Убедитесь, что у вас (или у того, кто будет собирать приложение) есть:
*   **Mac** с установленным **Xcode** (скачивается бесплатно из App Store).
*   **Аккаунт разработчика Apple** (Apple Developer Program) для публикации в App Store.

## 2. Перенос проекта
1.  Скопируйте всю папку с проектом `hemoday-mobile-MAIN-` с вашего Windows компьютера на Mac.
2.  Откройте **Terminal** на Mac и перейдите в папку проекта:
    ```bash
    cd /путь/к/папке/hemoday-mobile-MAIN-
    ```

## 3. Установка зависимостей (на Mac)
Выполните следующие команды в терминале, чтобы убедиться, что все на месте:

```bash
# Установка зависимостей проекта
npm install

# Синхронизация Capacitor (это также установит CocoaPods зависимости)
npx cap sync ios
```

> **Примечание:** Если команда выдаст ошибку, связанную с `CocoaPods`, убедитесь, что он установлен (`sudo gem install cocoapods`) и попробуйте снова.

## 4. Сборка в Xcode
1.  Откройте проект в Xcode следующей командой:
    ```bash
    npx cap open ios
    ```
    (Или откройте файл `ios/App/App.xcworkspace` вручную через Finder).

2.  В Xcode:
    *   В левой панели выберите **App** (самый верхний синий значок).
    *   Перейдите во вкладку **Signing & Capabilities**.
    *   В разделе **Team** выберите ваш аккаунт разработчика (Add Account, если его там нет).
    *   Убедитесь, что **Bundle Identifier** (`com.hemoday.app`) уникален и зарегистрирован в вашем аккаунте.

## 5. Настройка иконок и сплэш-скринов
Иконки и экраны загрузки уже были сгенерированы и помещены в проект. Вы можете проверить их в папке `App/Assets` в Xcode.

## 6. Публикация (Archive)
Для выгрузки в App Store Connect:
1.  В Xcode выберите в верхнем меню **Product** -> **Destination** -> **Any iOS Device (arm64)**.
2.  Выберите **Product** -> **Archive**.
3.  После завершения архивации откроется окно Organizer. Нажмите **Distribute App**.
4.  Выберите **App Store Connect** -> **Upload**.
5.  Следуйте инструкциям на экране для загрузки сборки.

## 7. App Store Connect
После загрузки сборки:
1.  Зайдите на [appstoreconnect.apple.com](https://appstoreconnect.apple.com).
2.  Создайте новое приложение (если еще нет).
3.  Заполните информацию (описание, скриншоты, возрастные ограничения).
4.  В разделе **Build** выберите загруженную версию.
5.  Отправьте на проверку (**Submit for Review**).

---
**Готово!** Теперь ваш проект полностью настроен для iOS, осталось только собрать его на Mac.
